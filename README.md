# 99tech - Technical Problems Solutions

This repository contains solutions to three technical problems:

## Projects

### [Problem 4](./problem_4/) - Sum to N Implementation
Multiple implementations to calculate the sum from 1 to n, with performance benchmarking capabilities.

**Key Features:**
- Three basic implementations (Iterative, Recursive, Mathematical Formula)
- Five alternative implementations
- Performance comparison and benchmarking
- TypeScript implementation

See [problem_4/README.md](./problem_4/README.md) for details.

### [Problem 5](./problem_5/) - CRUD Server
A production-ready backend server built with ExpressJS and TypeScript, implementing full CRUD operations.

**Key Features:**
- ExpressJS backend with TypeScript
- Full CRUD operations with PostgreSQL
- API authentication and validation
- Swagger/OpenAPI documentation
- Docker support
- Comprehensive testing

See [problem_5/README.md](./problem_5/README.md) for details.

### [Problem 6](./problem_6/) - Scoreboard Module
A real-time scoreboard system that displays the top 10 users' scores with live updates.

**Key Features:**
- Real-time scoreboard updates (top 10 users)
- Secure score update API with JWT authentication
- Rate limiting and abuse prevention
- WebSocket support for live updates
- PostgreSQL and Redis integration
- Docker support

See [problem_6/README.md](./problem_6/README.md) for details.
